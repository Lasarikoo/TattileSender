# Artefactos legacy

Esta carpeta está destinada a almacenar información del sistema heredado
(MossosWSNet.exe) y materiales de referencia.

- Binarios antiguos (`MossosWSNet.exe`, DLLs) y archivos de configuración usados
  previamente.
- Logs de ejecución y capturas de red de ejemplo.

**Importante:** los certificados `.pfx` y cualquier dato sensible **no** deben
versionarse ni copiarse a este repositorio. Para pruebas, se almacenarán fuera
 del control de código fuente en ubicaciones seguras.
